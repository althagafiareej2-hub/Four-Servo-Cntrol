# Four Servo Control

In this task, I used Tinkercad to control four micro servo motors using Arduino Uno. This project demonstrates servo movement from 0 to 180 degrees and returning to the center position.

I started by adding the project components, including Arduino Uno, jumper wires, and four Micro Servo Motors. Then, I connected the red wires of each servo to the 5V power pin on the Arduino, and the black wires to the GND pin. The signal wires were connected to the digital pins 3, 5, 6, and 9 on the Arduino.

After completing the connections, I wrote the control code using C++ to program the servo movements and tested the circuit through Tinkercad simulation.

After finishing the project, I added the code, output image, and video showing the result, then uploaded the files to GitHub.

## Tools Used
- Arduino Uno
- 4x Micro Servo Motors
- Tinkercad
- C++

## Skills Applied
- Designing electronic circuits using Tinkercad
- Writing control code using C++
- Connecting and programming servo motors

## Result

This project improved my skills in microcontroller programming and motor control. It also gave me more practice in connecting and programming servo motors using Arduino and creating simple motion systems.
